### The challenge

Users should be able to:

- Toggle extensions between **active** and **inactive** states
- Filter extensions by **All / Active / Inactive**
- Remove extensions from the list
- Toggle between **light and dark mode**
- View a responsive layout on different screen sizes
- See hover and focus states for interactive elements

### What I learned

In this project, I practiced:

- Structuring reusable UI components using semantic HTML
- Managing UI state with JavaScript (`data-*` attributes)
- Filtering content dynamically
- Implementing dark mode using CSS variables
- Using event delegation for better performance
- Saving user preferences with `localStorage`

This project helped me better understand how JavaScript interacts with the DOM in a real-world UI.
